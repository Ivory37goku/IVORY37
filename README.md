# 🎯 IVORY37 TOOLS COLLECTION

Colección de herramientas desarrolladas por IVORY37 para análisis y verificación de servicios IPTV y streaming.

---

## 📦 REPOSITORIO COMPLETO

Este repositorio contiene las siguientes herramientas:

### 1️⃣ **м3υ_√_ᴵⱽᴼᴿʸ³⁷ - IPTV Server Verifier**
> Archivo: `Ivory37M3U.py`

Herramienta de verificación y extracción de datos para servidores IPTV (XUI, Xtream UI, Xtream Codes).

**Características:**
- 🔍 Verificación de credenciales en servidores IPTV
- 🌐 Escaneo automático de espejos/dominios alternativos
- 🔄 Soporte para proxies (HTTP, SOCKS4, SOCKS5)
- 📊 Interfaz con estadísticas en tiempo real
- 🎯 Detección de contenido: Live TV, Películas, Series

---

### 2️⃣ **📦 Python Module Installer**
> Archivo: `modulos.py`

Gestor automático de instalación de módulos Python.

**Módulos que instala:**
- aiohttp, beautifulsoup4, cloudscraper, colorama
- fake-useragent, m3u8, requests, selenium
- y más de 25 librerías esenciales

**Características:**
- ✅ Verifica si los módulos ya están instalados
- 🔄 Actualiza pip automáticamente
- 📊 Resumen detallado de instalación
- 🎯 Instalación individual o masiva

---

### 3️⃣ **📺 AtresPlayer Tool**
> Archivo: `AtresPlayer_By_IVORY37.py`

Herramienta especializada para AtresPlayer con capacidades de automatización y extracción de contenido.

**Características:**
- 🔐 Automatización de login en AtresPlayer
- 📥 Descarga de contenido multimedia
- 🔍 Extracción de enlaces de streaming
- 📊 Gestión de cuentas y perfiles
- ⏱️ Programación de descargas
- 🎯 Soporte para múltiples calidades de video

---
